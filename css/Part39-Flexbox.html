Properti wadah flexsibel adalah:

  • flex-decoration. Value = [ column/column-reverse || row/row-reverse ]
  • flex-wrap. Value = [ wrap(untuk membuat element fleksibel)/ nowrap(default)/ wrap-revere ]
  • flex-flow. [ Properti singkatan untuk kedua flex-decoration dan flex-wrap ]
  • flex. value = 25%(Untuk besaran flex)
  • justify-content [ Membuat conten bisa di start/center/end
  • align-items. [ Membuat conten bisa di start/center/end ]
  • align-content. [ Membuat conten bisa di start/center/end ]
