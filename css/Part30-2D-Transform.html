Dengan properti CSS transform Anda dapat menggunakan metode 2D berikut
  1. tranlate()
  2. rotate()
  3. scaleX()
  4. scaleY()
  5. scale()
  6. skewX()
  7. skewY()
  8. skew()
  9. matrix()

tranlate() mereng ke kiri bawah atau ke kanan atas

rotate() membuat elemen berputar sesuai deg(derajat)

transform: scale() membuat elemen membesar lebar & tinggi seperti di Zoom

skew() membuat elemen menjadi condong

transform: matrix() membuat elemen bergeser ke kanan[bisa dimanipulasi sesuka hati]
