Properti Multi-Kolom CSS

Dalam bab ini kita akan memperlajari tentang properti multi-kolom berikut:

  • column-count [Properti column-count menentukan jumlah kolom yang harus dibagi menjadi elemen.]
  • column-gap [Properti column-gap menentukan celah antara kolom menggunakan pixel.]
  • column-rule-style [Properti column-rule-style menentukan gaya aturan antar kolom:]
  • column-rule-width [Properti column-rule-width menentukan lebar aturan antara kolom:]
  • column-rule-color [Properti column-rule-color menentukan warna aturan antar kolom:]
  • column-rule
  • column-span
  • column-width [Properti column-width menentukan lebar optimal yang disarankan untuk kolom.]
